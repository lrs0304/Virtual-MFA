# Virtual MFA v1.2

> 发布于 2026-06-30 · 上一版：`v1.1`

## 这一版主要做两件事：**深色模式** 和 **识别失败的体验**

如果你日常开着系统深色，主屏白闪闪的体验终于结束了；如果你扫码或相册导入时遇到识别不出 / 不是 MFA 配置的图，这次的兜底交互也更不"一笔带过"了。

## 更新内容

### 🌙 深色模式（跟随系统，默认开启）
- 应用界面会自动跟随系统的浅色 / 深色模式切换，无需手动操作
- 不想跟随？「设置 → 显示 → 跟随系统深色模式」一键关闭，永远保持亮色
- 深色下品牌蓝做了暗化处理，AppBar 不再"挖洞"；分区标题、链接文字用对比度更高的亮蓝，照样清晰

### 🔍 识别到非 MFA 二维码也能看原文
- 扫码或相册导入时，如果识别到的不是合法的 MFA 配置（otpauth），现在会**统一**跳转到原文预览页，让你看清楚扫到的到底是啥 + 一键复制
- 不需要这个中间页？「设置 → 显示 → 无效二维码显示原文预览」可以关掉，关掉后只弹一条 Toast

### 🖼️ 图片调整页体验修复
- 缩放后切到「调整」tab，缩放比例不再被重置——两个 tab 现在共享同一个预览
- 缩放模式下底部不再露出蓝色背景
- 状态栏 / 标题栏在两个页面（"重新识别" / 二维码内容预览）都不再叠加，刘海机型也对齐
- 这两个页面现在也加上了 `FLAG_SECURE`，截屏 / 录屏 / 最近任务缩略图都不会泄漏图片或解码原文

### ✨ 视觉小修
- 二维码内容预览页：去掉了卡片四个角莫名的灰色描边，改用更轻量的 1dp 边框
- 设置页滚动条样式与「关于」页统一，更细更克制
- 失败提示条 / 信息提示条在深色模式下也会自动用合适的深底亮字

## 体积 / 兼容性

- Release APK 体积与 v1.1 基本持平
- 数据完全兼容，账号 / 收藏 / 排序原地保留，直接覆盖安装即可
- 默认行为变化：**全新安装**时跟随系统深色模式开关是开的；从 v1.1 升级上来也是开（如不需要可手动关闭）
