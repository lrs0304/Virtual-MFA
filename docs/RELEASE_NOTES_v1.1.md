# Virtual MFA v1.1

> 发布于 2026-06-29 · 上一版：`v1.02`

## 这次主要解决一件事：**扫码扫不出**

把扫码引擎从 `zxing` 整体迁移到 **CameraX + Google ML Kit**，彻底修复了在 小米 15 Pro / 华为 Mate 60 Pro 等新机型上"对着二维码毫无反应"的问题。现在基本是**瞬间识别**。

## 更新内容

- **实时扫码重写**：换用 ML Kit，识别速度和成功率大幅提升
- **相册扫码同步升级**：跳过老的多轮二值化，长 URL 二维码也能稳定识别
- **识别失败可补救**：相册识别不出时，可在新增的图片调整页里 **缩放 / 调亮 / 二值化** 后重试
- **非 MFA 二维码也能看**：扫到普通二维码会展示原文 + 一键复制，不再"识别成功却什么都没发生"
- **新增「关于」页**：应用介绍 / GitHub 仓库 / 支付宝赞赏入口
- **Debug 包应用名加 `-dev` 后缀**：和正式包同机并存时一眼区分

## 体积 / 兼容性

- Release APK 约 **7.6 MB**，与上一版基本持平
- 数据完全兼容，账号 / 收藏 / 排序原地保留，直接覆盖安装即可
